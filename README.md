# Off3D Management UI 🚀

Bem-vindo(a) ao repositório do Front-end do **Off3D Studio**! Esta é uma aplicação desenvolvida em **Angular** para gerenciar as operações de um estúdio de manufatura aditiva (impressão 3D).

Este projeto é a interface oficial que consome a [Off3D Management API](https://github.com/Amandapvln/off3d-management-api).

---

## 🎯 Objetivo do Projeto
O objetivo desta UI é fornecer uma experiência fluida para:
* Gerenciar pedidos de impressão (Sales).
* Monitorar a fila de produção e status das máquinas (Manufacturing).
* Autenticação segura via JWT.

## 🛠️ Stack Tecnológica
* **Framework:** Angular 17+
* **Estilização:** SCSS (com suporte a Tailwind/Angular Material)
* **Gerenciamento de Estado:** RxJS
